## 📝 AI Text Summarizer with ROUGE Evaluation

This project is a web-based AI text summarizer built with **FastAPI** and a **PEGASUS** transformer model.  
Users can paste long text, generate an abstractive summary, and optionally compare it with a reference summary using **ROUGE-1**, **ROUGE-2**, **ROUGE-L**, and **ROUGE-Lsum** scores.
